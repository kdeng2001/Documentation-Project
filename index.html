Introduction

In CSS, properties are used to style HTML content. One such property is display. Two commonly used values for this property are Flexbox and Grid. By setting display to either of these values, the developer can adjust the layout of content through additional styling properties that depend on the display property's value. Though they are Flexbox or Grid are different methods, it is possible to achieve the similar styling results with either value. By the end of this documentation, you will have learned the basics of using Flexbox and Grid.

To understand using Flexbox and Grid, we should understand the two basic layout methods. They are Block and Inline. The Block layout method orients elements from top to bottom, while the Inline method orients elements from left to right. Certain HTML elements apply these methods by default. For example the list item element has the Block layout by default. However, these methods can also be set using the display property.

li { display: block; }
