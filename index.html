Introduction

In CSS, properties are used to style HTML content. One such property is display. Two commonly used values for this property are Flexbox and Grid. By choosing either of these values for display, the developer can adjust organize and adjust the orientation of content through additional styling properties that depend on the display property's value. Though they are Flexbox or Grid are different methods, it is possible to achieve the similar styling results with either value.

By the end of this documentation, you will have learned the basics of using Flexbox and Grid.
