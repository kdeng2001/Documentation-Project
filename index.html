Introduction

In CSS, properties are used to style HTML content. One such property is display. Two commonly used values for this property are Flexbox and Grid. By setting display to either of these values, the developer can adjust the layout of content through additional styling properties that depend on the display property's value. Though Flexbox or Grid are different methods, it is possible to achieve the similar styling results with either display value. By the end of this documentation, you will have learned the basics of using Flexbox and Grid.

To understand using Flexbox and Grid, we should understand the two basic layout methods. They are Block and Inline. The Block layout method orients elements from top to bottom, while the Inline method orients elements from left to right. Certain HTML elements apply these methods by default. For example the list item element has the Block layout by default. However, these methods can also be manually set with the display property.

li { display: block; }
Flexbox

The Flexbox has two main axis. The main axis is the direction child elements are placed. This can be set using the flex-direction property, after having set display to flex in a container element. Child elements of the flex container become flex items.

Another important property for flex:

  1. flex-wrap:

    This property can have values nowrap or wrap. When set to nowrap, the container will remain in a single row/column (depending on flex-direction). If the flex items' dimensions exceed the flex container's dimensions, the content will overflow. When set to wrap, if the flex items' dimensions exceed the flex container's dimensions, the content will "wrap" around to another line. This new line can be treated as a separate flex container with the same properties.

.flex-container { flex-wrap: nowrap; }
Flex Items

Here's some more important properties of flex. These apply to flex-items, and should be placed there.

  1. flex-grow:

    This property determines how much of the flex container's remaining space should be assigned to the flex item's main size. Flex-grow is particularly useful when the sum of the container items' size is smaller than the size of the container. Flex-grow is set inside the flex item. It's default value is 0.

  2. flex-shrink:

    This property determines how and which flex items shrink when the sum of all flex items' size is greater than the flex containers' dimensions. Like flex-grow, it's set inside the flex item and has a default value of 0.

  3. flex-basis:

    This property sets the initial size of the flex item.

  4. flex:

    This property is shorthand for flex-grow, flex-shrink, and flex-basis in this order. Using this property can help reduce the number of lines used, and ensure each value is set.

.flex-item { flex: 1, 0, 200px; }
Grid

Grids work similar to tables, but offer more varied layouts through properties set in child elements. There are many grid properties. These can influence all rows and columns or dimensions and behaviors of boxes of the grid, or (row number, column number).

Here are some of the grid properties.

  1. grid-template-columns

  2. grid-template-rows

  3. grid-template-areas

  4. grid-auto-columns

  5. grid-auto-rows

  6. grid-auto-flow

  7. grid:

    Shorthand for the above.

The grid properties below apply to boxes of a grid.

  1. grid-row-start

  2. grid-column-start

  3. grid-row-end

  4. grid-column-end

  5. grid-row

  6. grid-column

  7. grid-area

  8. row-gap

  9. column-gap

  10. gap

.grid-container { display: grid; }
